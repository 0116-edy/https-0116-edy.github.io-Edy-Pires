Como Fazer Upload do Arquivo HTML no GitHub

Guia passo a passo para publicar o site da A.A.M.E. SERVIÇOS LDA

1 Acesse seu repositório

Faça login na sua conta do GitHub e clique no repositório que você criou (seuusuario.github.io).

Dica: Se você ainda não criou um repositório, crie um com o nome seuusuario.github.io (substitua "seuusuario" pelo seu nome de usuário).

2 Clique em "Add file"

No seu repositório, clique no botão verde "Add file" e selecione "Upload files".

Importante: Certifique-se de que o arquivo principal tenha o nome index.html.

3 Arraste seu arquivo

Arraste o arquivo HTML da A.A.M.E. SERVIÇOS para a área indicada ou clique em "choose your files".

4 Adicione uma mensagem de commit

Na parte inferior da página, digite uma mensagem descritiva como "Site inicial da A.A.M.E. SERVIÇOS".

5 Clique em "Commit changes"

Finalize o upload clicando no botão verde "Commit changes".

6 Acesse seu site

Seu site estará disponível em: https://seuusuario.github.io

Nota: Pode levar alguns minutos para o site ficar disponível após o upload.

Resolução de Problemas Comuns

Site não atualiza?

Às vezes, o GitHub Pages pode demorar alguns minutos para atualizar. Tente:

  1. Limpar o cache do seu navegador (Ctrl+F5)
  2. Aguardar 5-10 minutos
  3. Verificar se o arquivo está na branch "main" ou "master"

Erro 404?

Se você recebe um erro 404:

  1. Verifique se o repositório tem exatamente o nome: seuusuario.github.io
  2. Confirme se o arquivo principal se chama index.html
  3. Verifique se o GitHub Pages está ativado em Settings > Pages

Dica Final

Depois de fazer upload do arquivo HTML, você pode personalizar ainda mais seu site:

  • Adicionar uma pasta "images" com fotos dos seus projetos
  • Incluir um arquivo CSS separado para estilos
  • Adicionar mais páginas (sobre.html, servicos.html, etc.)
